# Uppdateringar

Den här sidan dokumenterar hur uppdateringar bör hanteras.

## Före uppdatering

- Läs release notes för större Home Assistant-uppdateringar.
- Kontrollera om YAML-syntax eller integrationer påverkas.
- Ta backup.
- Kontrollera om viktiga automationer är beroende av integrationer som ändras.

## Efter uppdatering

- Kontrollera loggar.
- Kontrollera dashboards.
- Kontrollera kritiska sensorer.
- Kontrollera Zigbee2MQTT, MQTT och AppDaemon.
- Dokumentera eventuella problem och lösningar.
