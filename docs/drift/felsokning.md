# Drift och felsökning

Den här sidan är den praktiska startsidan när något inte fungerar.

## Snabb kontrollordning

1. Kontrollera Home Assistant-loggen.
2. Kontrollera om problemet uppstod efter omstart eller uppdatering.
3. Kontrollera berörda integrationer.
4. Kontrollera MQTT om sensordata saknas.
5. Kontrollera AppDaemon om Python-logik påverkas.
6. Kontrollera dashboards sist, eftersom de ofta bara visar följdfel.

## Dokumentera återkommande problem

Skapa gärna en egen sida eller ett avsnitt när samma fel återkommer. Skriv:

- symptom
- orsak
- hur felet upptäcks
- hur det åtgärdas
- hur det kan förebyggas
