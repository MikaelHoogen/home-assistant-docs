# Beslutslogg

Beslutsloggen används för att dokumentera viktiga vägval.

## Syfte

Beslutsloggen ska svara på:

- vilket beslut togs?
- varför togs det?
- vilka alternativ övervägdes?
- när bör beslutet omprövas?

## Exempel på beslut

- repo-struktur
- relation mellan Home Assistant och Hydromet
- elprisstrategi
- datalagring för mätdata
- större dashboard-principer
