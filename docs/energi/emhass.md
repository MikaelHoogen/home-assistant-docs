# EMHASS

Den här sidan dokumenterar hur EMHASS används i Home Assistant-systemet.

## Syfte

EMHASS används för optimering och beslutsstöd kopplat till elpris, solproduktion, batteri och styrbara laster.

## Att dokumentera

- indata till EMHASS
- prognoser
- prisserier
- batteridata
- styrbara laster
- vilka sensorer som används i Home Assistant
- hur resultat presenteras i dashboards

## Risker

- fel tidsupplösning mellan prisdata och EMHASS
- saknade prognosvärden
- sensorer som byter namn
- optimering som bygger på gammal prislogik
