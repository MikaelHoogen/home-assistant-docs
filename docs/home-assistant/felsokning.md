# Felsökning i Home Assistant

Den här sidan är en startsida för felsökning av Home Assistant-specifika problem.

## Grundkontroller

- Är Home Assistant igång?
- Är Mosquitto igång?
- Är Zigbee2MQTT igång?
- Finns fel i loggen?
- Har någon integration inte hunnit starta efter omstart?

## Dokumentera fel som återkommer

När ett problem uppstår flera gånger bör det dokumenteras med:

- symptom
- berörda entiteter
- sannolik orsak
- kontrollsteg
- åtgärd
- om problemet kan uppstå efter omstart

## Exempel på områden

- elprisdata saknas
- Zigbee-enheter svarar inte
- MQTT-sensorer uppdateras inte
- AppDaemon-app har stannat
- dashboard visar gamla värden
