# Packages

Home Assistant-konfigurationen bör organiseras i packages per funktion, rum eller område.

## Syfte

Packages gör det lättare att samla sensorer, helpers, templates och automationer som hör ihop.

## Dokumentera per package

För viktiga packages bör följande dokumenteras:

- syfte
- viktiga entiteter
- beroenden
- dashboards som använder entiteterna
- risker vid ändring
- felsökning

## Exempel på struktur

```markdown
# Package: energi

## Syfte

## Viktiga filer

## Viktiga entiteter

## Beroenden

## Felsökning
```
