# Dashboards

Den här sidan beskriver principer för dashboards i Home Assistant.

## Mål

Dashboards ska vara praktiska, tydliga och skilja på:

- driftstatus
- styrning
- analys
- diagnostik

## Dokumentera per dashboard

För varje viktig dashboard bör följande beskrivas:

- syfte
- målgrupp
- viktiga kort
- vilka entiteter som är kritiska
- om dashboarden är styrande eller bara visande

## Exempel

- Energi-dashboard
- Hydromet-dashboard
- Diagnos-dashboard
- Rumsspecifika dashboards
